## 关键决策记录

| 议题 | 决策 | 理由 |
|------|------|------|
| 反馈渠道 | 首页入口走 `mailto` 纯客户端方案 | 零成本零依赖；邮件从用户真实邮箱发出，可直接回复对话 |
| 内容存储 | 讲解落 PostgreSQL，音频上 R2 | 杜绝缓存过期/重部署导致的重复 AI 费用 |
| 内容正确性 | 事实从 Wikidata，叙述用 grounded generation | 从源头减少幻觉，审核负担降到只剩抽查 |
| 是否展示藏品图 | 是，用公有领域参考图 | 确认识别结果、探索页可用、离线价值 |
| 数据模型 | 现在就泛化为通用展品，暂只填绘画 | 改 schema 便宜，retrofit 贵（YAGNI 平衡） |
| 识别返回 | top-3 + 置信度 | 避免"自信报错答案"，提升体感准确率 |
| 语种 | 中 + 英 + 法 | 奥赛是法国馆；每多一语种内容成本与审核翻倍 |
| 收费时机 | 内容完整度 ≥70% 后再开启 | 现在收费转化低、易招差评 |
