## 1. 设计目标与原则
- **简洁直观**：一眼聚焦“拍照识别讲解”核心能力。
- **文化气质**：低饱和古典色 + 半透明浮层 + 优雅衬线字体。
- **一致性**：iOS/Android 双端一致，平板适配。
- **可扩展**：模块化组件，可快速切换/新增功能与语言。
- **可验证**：关键路径在 3 步内完成，录屏可回放测试。

视觉基调：
- 主色：炭黑（#2C2C2C）与金褐（#C1A46D）
- 背景：象牙白（#F7F4EF）
- 强调：识别 CTA 使用高亮黄（#FFCC00）

## 2. 信息架构（IA）
**底部导航（5 项）**：Home | Explore | （中置）Capture | Footprints | Settings

- **登录/注册**：启动时、受限操作或设置页进入（邮箱/Google/Facebook/（iOS）Apple）。
- **语言切换**：在设置页切换**整个 App**语言，实时刷新（非弹层临时切换）。
- **讲解页**：文字内容始终可见，耳机连接时默认自动**播放 TTS**；提供播放/暂停按钮。
- **AI 问答**：支持文字输入与语音输入（长按或点击录音），答案可文字+可选 TTS。
- **订阅弹框**：周/月/年三档，权益对比 + 试用提示 + 恢复购买。

已移除：禁止拍照模式、离线包页面（本期不做，入口隐藏）。

## 3. 用户旅程与核心流程

### 3.1 新用户首启
1) 启动横幅：品牌 Logo + 一句话价值主张；
2) 申请权限（相机/麦克风/定位，按需与时机弹出）；
3) 首页中置“拍照识别讲解”按钮。
> *可选*：功能受限时弹订阅/登录引导。

### 3.2 拍照识别
- 进入相机 → 拍摄 → 识别动画 → 结果卡（作品名/作者/置信度/简介）
- 低置信度：展示 2–3 候选卡片供选择

### 3.3 讲解页（Explanation）
- 头图 + 标题信息
- 结构化文字（基本信息→看点→延展）
- **播放条**：播放/暂停、进度、重听（耳机连接时默认自动播放）
- **问答区**：文字输入 + 语音按钮（按住说话/点击开始结束）

### 3.4 订阅触发（示例）
- 免费额度用尽；访问高级音色/极速通道；足迹云同步等 → 弹出订阅 Modal

### 3.5 账户登录/注册
- 触发点：设置页、订阅、云同步
- 支持邮箱、Google、Facebook、Apple（仅 iOS 展示）

## 4. 页面详设

### 4.1 首页 Home
- 顶部：Logo + 定位城市
- 中部：**大圆 CTA「📷 拍照识别讲解」**（呼吸动效）
- 下方：附近博物馆横滑 + 今日精选展品瀑布流
- 顶部提醒：未连接耳机时提醒“连接耳机可用语音讲解”

### 4.2 拍照识别 Capture
- 相机取景 + 圆形快门
- 底部：识别历史入口
- 拍照后：识别动画 → 结果卡 + 进入讲解页

### 4.3 讲解页 Explanation
- 头图 + 标题（作品名/作者/馆名）
- **播放条**（始终可见）：播放/暂停、进度、时长
- 文本讲解分段可折叠
- **AI 问答区**：输入框 + 发送；右侧语音按钮（长按/点击录音）

### 4.4 探索 Explore
- 顶部搜索框（城市/馆/作品）
- 结果列表：馆卡（封面/城市/开放与票价徽章）
- 馆详情：开放时间、票价、交通、简介、代表作 Top5、打开地图

### 4.5 足迹 Footprints
- 时间线视图（日期分组）
- 卡片：缩略图 + 作品名 + 馆名 + 时间 + 收藏
- 点击进入讲解页；支持搜索/筛选

### 4.6 设置 Settings
- 账户：登录/登出/删除账户
- **语言**：App 语言切换（即时生效）
- 通用：通知、权限、清缓存
- 支持：评分、帮助、联系我们
- 法务：隐私/条款；关于：版本/分享

### 4.7 登录/注册 Auth
- SSO：Google / Facebook /（iOS）Apple
- 邮箱登录/注册（邮箱 + 验证码/密码）
- 法务：隐私/条款链接

### 4.8 订阅弹框 Subscription Modal
- 顶部标题「开启无限聆听」
- 选择：周 / 月 / 年（ToggleButtons）
- 权益说明卡片：不限识别、极速回答、精选音色…
- CTA：开始订阅；辅助：恢复购买；取消 ×

## 5. 组件与模式（Components & Patterns）
- **Buttons**：主按钮（金褐/黑字）、次按钮（白底描边）
- **Cards**：作品卡、馆卡、结果卡
- **Nav**：中置圆形拍照 FAB + BottomAppBar 缺口
- **Modals**：订阅 Modal、登录 BottomSheet
- **Inputs**：搜索框、问答输入框、语音按钮（长按/点击）
- **Feedback**：识别波纹、TTS 声波、加载骨架屏、Toast

动效建议：
- CTA 呼吸；识别波纹；TTS 声波；问答逐字流式

## 6. 多语言与本地化（i18n）
- App 级语言切换（设置页）→ 立即覆盖 UI 文案
- 语言包：`assets/i18n/*.arb`
- 文字长度膨胀与 RTL（后续版本）
- 日期/价格格式化本地化

---

## 7. 听觉体验（TTS）与语音问答（VQA）
- 耳机已连接：默认自动播放；但文字区始终可见
- 播放条：播放/暂停/进度/重听
- 语音问答：录音动画/峰值；提交后**优先返回文字**，可配置同时生成语音回答

---

## 8. 订阅与付费（Subscription）
- 周 / 月 / 年 SKU；恢复购买；试用条款
- 触发策略：额度用尽/高峰线路/高阶音色等
- 法务：价格展示规则、取消方式、自动续订说明

---

## 9. 可访问性（A11y）
- 对比度、字体缩放、屏幕阅读器标签
- 语音控件可聚焦/可读
- 动效可减少（系统“减少动态效果”跟随）

---

## 10. 埋点与指标（Analytics）
- 首启、授权通过/拒绝
- 拍照→识别成功/失败、耗时
- 讲解播放/暂停、问答使用（文字/语音）
- 订阅查看/点击/支付成功
- 语言切换、登录来源（邮箱/SSO）

---

## 11. Figma Tokens 与工程落地
- Tokens 存放：`tokens/gomuseum.tokens.json`
- 主题生成：脚本将 Tokens 映射到 Dart 常量（`lib/theme/`）
- 组件引用统一使用 Theme/Token，避免硬编码色值

## 12. 原型建议与文件结构
**Figma 层级**
```
GoMuseum
├── Tokens
├── Components
│   ├── Buttons / Cards / Modals / Inputs / Nav
├── Pages
│   ├── Home / Capture / Explanation / Explore / Footprints / Settings / Auth / Subscription
└── Prototype
    ├── First-run Flow / Capture Flow / Subscription Flow
```

**代码结构（建议）**
```
lib/
├── theme/ (tokens.dart, app_theme.dart)
├── ui/ (root_nav.dart, home_page.dart, explore_page.dart, capture_page.dart, footprints_page.dart, settings_page.dart, explanation_page.dart)
└── widgets/ (subscription_modal.dart, auth_sheet.dart, ...)
assets/i18n/ (en.arb, zh.arb, ...)
tokens/ (gomuseum.tokens.json)
```
